
## Beaker library locations

#### Linux:
 * /usr/share/beaker
 * /usr/local/share/beaker
 * {sys.prefix}/share/beaker
 * ~/.local/share/beaker
 * {os.environ["XDG_DATA_HOME"]}/beaker

#### Mac OSX:

 * /usr/share/beaker
 * /usr/local/share/beaker
 * ~/.local/share/beaker
 * ~/Library/Beaker

#### Windows:
 * %PROGRAMDATA%\beaker
 * %APPDATA%\beaker
 * %LOCALAPPDATA%\beaker
